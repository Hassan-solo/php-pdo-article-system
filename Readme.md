#  Article Manager – Simplified Back Office

##  Project Description

Article Manager is a simple CRUD web application built with PHP and MySQL that allows an admin to:

- View all articles 
- Add a new article 
- Edit an existing article  
- Delete an article 

The project is built using **Object-Oriented PHP (OOP)** and **PDO with prepared statements** to ensure security and prevent SQL injection.

---

## Technologies Used

- PHP (OOP)
- MySQL
- PDO (Prepared Statements)
- HTML / CSS
- XAMPP (Local Server)
